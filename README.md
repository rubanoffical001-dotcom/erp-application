## How to Use

1. **Open the Application**
   - Simply open `index.html` in any modern web browser
   - No server or installation required

2. **Navigate**
   - Click on any card on the home page to access that module
   - Use the "Back to Home" button to return to the dashboard

3. **Add Records**
   - Fill in the form fields
   - Calculations update automatically as you type
   - Click "Save" to store the record
